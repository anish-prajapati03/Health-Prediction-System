# 🏥 Health Prediction System (AI/ML + Django)

## 📌 Project Overview

The **Health Prediction System** is an AI-powered web application that predicts possible diseases based on user-input symptoms.
It combines **Machine Learning models** with a **Django backend** to provide fast and accurate health insights.

---

## 🚀 Live Demo

🔗 **Try the App:** https://health-prediction-system-ug1k.onrender.com/


---

## 🎯 Features

* 🧠 Predict diseases using AI/ML models
* 📝 User-friendly symptom input interface
* ⚡ Fast and real-time prediction
* 🔐 Secure backend using Django
* 📊 Scalable architecture for healthcare applications

---

## 🛠️ Tech Stack

### 👨‍💻 Backend

* Python
* Django

### 🤖 Machine Learning

* Scikit-learn
* Pandas
* NumPy

### 🎨 Frontend

* HTML, CSS

---

## ⚙️ How It Works

1. User enters symptoms
2. Data is processed and formatted
3. ML model predicts disease
4. Result is displayed to user

## 🔧 Installation & Setup

### 1️⃣ Clone the Repository

```
https://github.com/anish-prajapati03/Health-Prediction-System/

```

### 2️⃣ Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the Application

#### For Django:

```
python manage.py runserver
```
